Since way back in senior year of highschool, when I decided I would learn JavaScript to [make a minecraft bot](https://github.com/PrismarineJS/mineflayer). Then I learned TypeScript so I could [make APIs](https://github.com/u9g/minecraft-proxy-handler/commit/6caa09e3fbcbcf3b0e5d96054e4ff46561d4233f) for myself and others. Around the same time I also started learning Java because if you're already making minecraft bots you may as well start [making minecraft mods](https://github.com/u9g/ReverseHopper/commit/36300a67a2ecb2400e9fd300634fc91b3e2434bd) and [contributing to Java tools](https://github.com/u9g/McDataExtracting/commit/2e68722a6afb4366bd1fdb0f12fb8afa127f9fef) for JavaScript minecraft bots. Since then, I learned Kotlin, which is the TypeScript to JavaScript in the Java world, and continued making [minecraft mods](https://github.com/u9g/cosmicsky-client) and [minecraft plugins](https://github.com/u9g/Minigames). Around this time, I took a look around the programming landscape and tried out Gleam, which is a growing programming language that compiles to Erlang, or in my use case JavaScript, for [more minecraft mods](https://github.com/u9g/thatgleammod).

After that, I started looking into other things which I found interesting, like data presentation formats and databases and stumbled upon Trustfall. Trustfall is, according to the github README, *"A query engine for any combination of data sources. Query your files and APIs as if they were databases!"*. And that's what I did for a while, until I wanted to expand the project even more, being a programmer myself, I created a [github adapter](https://github.com/u9g/github_adapter) and [contributed back to the original project](https://github.com/obi1kenobi/trustfall/commits?author=u9g) under the guidance of the extremely patient and helpful maintainer.

Later, I took my Rust knowledge on the road when I got interested in learning about linters and a brand new project at that time, OXC, which describes itself as, *"A collection of high-performance JavaScript tools."*. I found this premise interesting, and I also found Rust to be a great language for new contributors to a project, so I [spent a good amount of time contributing to this project too](https://github.com/oxc-project/oxc/commits?author=u9g).
